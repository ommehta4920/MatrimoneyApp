# Matrimony App (Android)

A full-featured Android Matrimony Application built using Java/Kotlin and Gradle.
The app allows users to create profiles, set preferences, find matches, and communicate securely.

## Features
- User registration & authentication
- Profile creation and management
- Matchmaking based on preferences
- Chat messaging between matched users
- Interest management
- Secure data handling

## Tech Stack
- Android (Java/Kotlin)
- Gradle
- REST API (Backend-ready)
- MySQL (Database design included)

## Database Design
The project includes:
- Users
- Profiles
- Preferences
- Matches
- Messages
- Interests

Refer to the **Matrimony Data Dictionary** for schema details.
